To integrate authentication into an {CLIENT} using {UIK}:

. *Set the URL of the server your {CLIENT} retrieves tokens from*
+
In `<project_name>/src/App.js`, locate the `rtcProps` declarion and replace `token : ''` with the following code:
+
[source, javascript]
----
// The base URL to your token server. For example, https://app-name.herokuapp.com"
tokenUrl : ''
----


{COMPANY} {UIK} automatically fetches a token from the token server at `tokenUrl`.
