<PlatformWrapper platform="web">
. In your {CLIENT}, Update the `appID` in the declarations to the value from Agora Console.

. Set the `token` variable to an empty string in the declarations.

. Set `channel` to the same string you specified in the web demo app.

. Update `tokenUrl` to the base URL of your token server. For example, `\https://<app-name>.herokuapp.com`.

. Start the dev server.
+
In the terminal, execute the following commands.
+
[source, terminal]
----
cd <app-name>
npm start
----
+
Use the URL displayed in the terminal to open the app in your browser.
